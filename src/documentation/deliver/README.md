# Completed Deliverables

This directory contains all completed tasks that have been reviewed and approved.

## Structure

Tasks are organized by sprint:
```
deliver/
├── sprint-1/
├── sprint-2/
├── ...
└── sprint-N/
```

## Completion Criteria

A task moves here when:
1. All acceptance criteria are met
2. Code review approved (if applicable)
3. Tests passing
4. Documentation updated
5. Approved by reviewer

## Archive Process

1. Move task file from `agile.role/[assignee]/` to `deliver/sprint-X/`
2. Update task status to DONE
3. Record actual hours
4. Add review notes
5. Update sprint metrics
